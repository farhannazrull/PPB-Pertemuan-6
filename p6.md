<div class="step"><strong>Langkah 2:</strong> Buat file <code>CalculatorViewModel.kt</code> yang mengextend <code>ViewModel</code>. Definisikan state untuk layar tampilan, angka pertama, dan operator yang sedang aktif.</div>

<code class="mono">class CalculatorViewModel : ViewModel() {
    var displayValue by mutableStateOf("0")
        private set
    var firstNumber by mutableStateOf(0.0)
        private set
    var pendingOperator by mutableStateOf("")
        private set
    var isNewInput by mutableStateOf(true)
        private set

    fun onNumberClick(number: String) {
        if (isNewInput) {
            displayValue = number
            isNewInput = false
        } else {
            displayValue = if (displayValue == "0") number else displayValue + number
        }
    }

    fun onOperatorClick(operator: String) {
        firstNumber = displayValue.toDoubleOrNull() ?: 0.0
        pendingOperator = operator
        isNewInput = true
    }

    fun onEqualsClick() {
        val secondNumber = displayValue.toDoubleOrNull() ?: 0.0
        val result = when (pendingOperator) {
            "+" -> firstNumber + secondNumber
            "-" -> firstNumber - secondNumber
            "x" -> firstNumber * secondNumber
            "/" -> if (secondNumber != 0.0) firstNumber / secondNumber else Double.NaN
            else -> secondNumber
        }
        displayValue = formatResult(result)
        pendingOperator = ""
        isNewInput = true
    }

    fun onClearClick() {
        displayValue = "0"
        firstNumber = 0.0
        pendingOperator = ""
        isNewInput = true
    }

    fun onDecimalClick() {
        if (isNewInput) {
            displayValue = "0."
            isNewInput = false
        } else if (!displayValue.contains(".")) {
            displayValue += "."
        }
    }

    private fun formatResult(value: Double): String {
        return if (value == value.toLong().toDouble()) {
            value.toLong().toString()
        } else {
            value.toString()
        }
    }
}</code>

fun CalculatorDisplay(value: String, modifier: Modifier = Modifier) {
    Box(
        modifier = modifier
            .fillMaxWidth()
            .background(Color(0xFF1C1C1E))
            .padding(horizontal = 24.dp, vertical = 32.dp),
        contentAlignment = Alignment.CenterEnd
    ) {
        Text(
            text = value,
            fontSize = if (value.length > 9) 32.sp else 52.sp,
            color = Color.White,
            fontWeight = FontWeight.Light,
            maxLines = 1
        )
    }
}</code>

<div class="step"><strong>Langkah 4:</strong> Buat composable <code>CalculatorButton</code> yang bisa dikustomisasi warna dan ukurannya.</div>

<code class="mono">@Composable
fun CalculatorButton(
    label: String,
    backgroundColor: Color,
    textColor: Color = Color.White,
    modifier: Modifier = Modifier,
    onClick: () -> Unit
) {
    Box(
        modifier = modifier
            .padding(4.dp)
            .aspectRatio(1f)
            .clip(CircleShape)
            .background(backgroundColor)
            .clickable { onClick() },
        contentAlignment = Alignment.Center
    ) {
        Text(
            text = label,
            fontSize = 24.sp,
            color = textColor,
            fontWeight = FontWeight.Medium
        )
    }
}</code>

<div class="step"><strong>Langkah 5:</strong> Buat composable <code>CalculatorScreen</code> yang menyusun tombol-tombol dalam grid 4 kolom.</div>

<code class="mono">@Composable
fun CalculatorScreen(viewModel: CalculatorViewModel = viewModel()) {
    val buttons = listOf(
        listOf("AC", "+/-", "%", "/"),
        listOf("7", "8", "9", "x"),
        listOf("4", "5", "6", "-"),
        listOf("1", "2", "3", "+"),
        listOf("0", ".", "=")
    )

    val operatorColor = Color(0xFFFF9F0A)
    val functionColor = Color(0xFFA5A5A5)
    val numberColor = Color(0xFF333333)

    Column(
        modifier = Modifier
            .fillMaxSize()
            .background(Color.Black)
    ) {
        CalculatorDisplay(
            value = viewModel.displayValue,
            modifier = Modifier.weight(1f)
        )
        Column(modifier = Modifier.padding(8.dp)) {
            buttons.forEach { row ->
                Row(modifier = Modifier.fillMaxWidth()) {
                    row.forEach { label ->
                        val bgColor = when (label) {
                            "AC", "+/-", "%" -> functionColor
                            "/", "x", "-", "+", "=" -> operatorColor
                            else -> numberColor
                        }
                        CalculatorButton(
                            label = label,
                            backgroundColor = bgColor,
                            modifier = Modifier.weight(if (label == "0") 2f else 1f),
                            onClick = {
                                when {
                                    label == "AC" -> viewModel.onClearClick()
                                    label == "=" -> viewModel.onEqualsClick()
                                    label == "." -> viewModel.onDecimalClick()
                                    label in listOf("+", "-", "x", "/") ->
                                        viewModel.onOperatorClick(label)
                                    else -> viewModel.onNumberClick(label)
                                }
                            }
                        )
                    }
                }
            }
        }
    }
}</code>

<h2>Penjelasan Kode</h2>

<h3>ViewModel sebagai Penyimpan State</h3>
<p>Dengan menempatkan <code>displayValue</code>, <code>firstNumber</code>, dan <code>pendingOperator</code> di dalam <code>CalculatorViewModel</code>, state tidak akan hilang ketika layar dirotasi. ViewModel hidup selama komponen UI yang memilikinya masih aktif. Properti dengan <code>private set</code> memastikan hanya ViewModel yang bisa mengubah nilainya, sementara UI hanya membaca.</p>

<h3>Logika Operasi dengan when</h3>
<p>Fungsi <code>onEqualsClick()</code> menggunakan ekspresi <code>when</code> untuk memilih operasi matematika berdasarkan nilai <code>pendingOperator</code>. Penanganan pembagian dengan nol dilakukan secara eksplisit menghasilkan <code>Double.NaN</code>, yang kemudian dapat ditampilkan sebagai pesan kesalahan.</p>

<h3>CalculatorButton yang Fleksibel</h3>
<p>Tombol "0" mendapat <code>weight(2f)</code> sehingga lebarnya dua kali tombol lain, meniru tampilan kalkulator standar. Penggunaan <code>aspectRatio(1f)</code> pada tombol lain memastikan setiap tombol berbentuk lingkaran sempurna tanpa perlu menentukan ukuran secara manual.</p>
